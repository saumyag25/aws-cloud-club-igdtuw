AWS Cloud Club is coming to IGDTUW

Get ready for an exciting and insightful journey with AWS

Through AWS cloud club we wish to explore the world of cloud computing as it is a new and emerging field and AWS is the most used cloud computing services. Students can gain more industrial experience working with AWS and AWS services.

AWS cloud club offers some exciting benefits, such as AWS credits, certification vouchers, exclusive experiences in partnerships with AWS employees, and opportunities for speaking and career related workshops

Students can deploy and integrate their projects using various AWS services and can use to make an impressionable resume.

There will be free credits, vouchers and some amazing goodies for its members along with various speaker sessions getting organized.

About AWS Cloud Club IGDTUW

AWS Cloud Clubs are student-focused, peer-driven user groups for post-secondary-level students worldwide. Run by Cloud Club Captains across the globe. The mission is to help more students expand their career options, increase their technical social circles, and discover ways to engage with AWS as a technical platform.

About AWS

Amazon Web Services(AWS) is a cloud computing platform which aims to provide low cost and on the go services to its customer. It includes a mixture of infrastructure-as-a-service(IaaS), platform-as-a-service(PaaS) and packaged-software-as-a-service(SaaS) offerrings. It has over 200 fully featured services from data centres globally.
